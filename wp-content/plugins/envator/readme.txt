Envator plugin display yours envato items in pretty widget.

== Description ==

This plugin display yours envato items in pretty widget

Sell or referrer envato items with your wordpress blog. Its too simple.

Please notify us if you find any bugs or issues with this widget as we are always trying to make
sure we provide the best applications and tools for this online.

**Features**

* Simple to use
* Easy to install
* Customizable
    
**Usage**

Once you have installed the sidebar widget you need to go to your widgets section of your blog and simply drag and drop the sidebar widget into the sidebar you wish to have it displayed in. Click the arrow to set your options and configure your Envator settings and you should be go to go.


== Installation ==

Either use your install option in the administration section of your WordPress blog or unzip
the files and drag and drop the entire folder envator into your plugins folder using your favorite ftp software.
